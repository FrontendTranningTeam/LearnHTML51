I. CSS/ Components

  1. Containers
    • '.container'
      • >= 1200px: 1140px
      • >= 992px: 960px
      • >= 768px: 720px
      • >= 576px: 540px
      • < 576px: None
      • Gutter width (khoảng cách giữa 2 cột): 30px (15px mỗi bên)
        Nếu không muốn có gutter: Đặt '.no-gutters' vào '.row'
    • 'containter-fluid': full-width
  2. Responsive Breakpoints

    @media (min-width: 576px) { ... }: phones

    @media (min-width: 768px) { ... }: tablet

    @media (min-width: 992px) { ... }: desktops+

    @media (min-width: 1200px) { ... }: large desktops+

  3. Grid System:
    • Grid system trong Bootstrap được chia làm 12 cột
    • '.row' phải được đặt trong '.container' hoặc '.container-fluid'
    • Nội dung đặt bên trong mỗi cột và mỗi cột phải là thẻ con ngay sau '.row'
    • Grid option: Có 4 loại cột tương ứng với 4 kích cỡ màn hình khác nhau
      • 'col-' < 576px
      • 'col-sm-' >= 576px
      • 'col-md-' >=768px
      • 'col-lg-' >=960px
      • 'col-xl-' >=1200px
      • Nesting column: Yes. Như Bootstrap 3
      • Column Ordering: Yes. Như Bootstrap 3
    • Auto-layout columns - Tự động layout cho cột
      • Equal width (Width bằng nhau): Đặt '.col' trong '.row' các col sẽ tự động có width bằng nhau
      • One column width (Width cho 1 cột): Đặt '.col-number' trong '.row' '.col-number' này sẽ có width tương ứng với number, các col còn lại sẽ tự động có width bằng nhau
      • Variable width content (Đặt biến cho width): col-{breakpoint}-auto

        • See the Pen [BS4] Auto-layout columns by Thang (@thang21) on CodePen.

    • Alignment
      • Vertical (chiều dọc)
        • align-items-start
        • align-items-center
        • align-items-end
      • Horizontal (chiều ngang)
        • justify-content-start
        • justify-content-center
        • justify-content-end
        • justify-content-around
        • justify-content-between

        • See the Pen [BS4] Alignments by Thang (@thang21) on CodePen.


    • Reordering Dùng '.order-{breakpoint}-number': Trình tự từ số nhỏ tới lớn

      See the Pen [BS4] Reordering by Thang (@thang21) on CodePen.

    • Image responsive: Nếu ở BS3 thì có class '.img-container' thì sang BS4 dùng '.img-fluid' với hiệu ứng tương tự
    • Icon: Nếu ở BS3 thì có icon là Glyphicons, thì sang BS4, bộ icon này không còn được hỗ trợ, vì vậy nếu muốn dùng icon thì phải thêm các thư viện khác

III. Cải tiến của Bootstrap 4

  • BS4 sử dụng flexbox
  • BS4 có thêm một vài component mới như: 'card', 'tooltip', 'scrollspy'
  • Break point của BS3 khác với BS4, BS4 có thêm điểm nhỏ hơn 768px

I. CSS/ Components

  1. Containers
    • '.container'
      • >= 1200px: 1170px
      • >= 992px: 970px
      • >= 768px: 750px
      • < 768px: None (auto)
      • Gutter width (khoảng cách giữa 2 cột): 30px (15px mỗi bên)
    • 'containter-fluid': full-width
  2. Grid System:

      See the Pen Grid System by Thang (@thang21) on CodePen.

    • Grid system trong Bootstrap được chia làm 12 cột
    • '.row' phải được đặt trong '.container' hoặc '.container-fluid'
    • Nội dung đặt bên trong mỗi cột và mỗi cột phải là thẻ con ngay sau '.row'
    • Grid option: Có 4 loại cột tương ứng với 4 kích cỡ màn hình khác nhau
      • 'col-xs-' < 768px
      • 'col-sm-' >= 768px
      • 'col-md-' >=992px
      • 'col-lg-' >=1200px
    • Offsetting Columns

      See the Pen oEgjvj by Thang (@thang21) on CodePen.

    • Nesting columns:

      See the Pen mXyeGp by Thang (@thang21) on CodePen.

    • Column ordering:

      See the Pen Column Ordering by Thang (@thang21) on CodePen.

  3. Images:
    • Responsive images:

      '.img-responsive': max-width: 100%, height: auto, display: block.

    • Image shapes:

      '.img-rounded'

      '.img-circle'

      '.img-thumbnail'

  4. Căn giữa một khối

    '.center-block': display: block; margin-left: auto; margin-right: auto;

  5. Clearfix

    Thuộc tính '.clearfix' dùng để xóa các float.

  6. Responsive embed (responsive nhúng)

    See the Pen Responsive embed by Thang (@thang21) on CodePen.

  7. Glyphicons

    Bootstrap 3 có hỗ trợ icon bằng Glyphicons: 'glyphicon glyphicon-align-left'

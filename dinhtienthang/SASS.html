I. Đặc tính của SASS

  • Varibles - Biến: $tenbien: thuộc tính

    See the Pen GQReRm by Thang (@thang21) on CodePen.

  • Nesting:

    Nesting được hiểu là sẽ viết css theo parent - child. Những thẻ thuộc child sẽ nằm bên trong thẻ parent.
    NOTE: Nếu nest quá nhiều thì sẽ dẫn đến tình trạng css bị quá tải khiến tốc độ tải trang chậm hơn so với bình thường. => mỗi một module là tên 1 class riêng biệt, tránh nesting quá nhiều.

  • Partials - Import:

    Partials được hiểu là chia nhỏ, phân nhánh. Dùng dấu gạch dưới "_" để đặt tên cho các file nhỏ. Và sau cùng là dùng @import để tập hợp các file partials và khi render ra CSS, nó sẽ được tập hợp vào cùng 1 file

  • Mixins:

    Thuộc tính Mixins là thuộc tính cho phép coder dùng lại 1 đoạn code mà không phải viết đi viết lại nhiều lần. Ta có thể truyền vào 1 biến bên trong mixin và khi dùng thì có thể thay đổi biến tùy ý muốn.
    Để tạo mixin: @mixin mixin_name($variable) {...}
    Khi muốn chèn mixin vào một class nào đó, thêm @include mixin_name($varible);

    See the Pen Mixins by Thang (@thang21) on CodePen.

  • Placeholder:

    Sử dụng @extend cho phép chúng ta chia sẻ 1 đoạn CSS từ selector này sang selector khác.
    Để tạo placeholder: %extend_name {...}
    Khi muốn chèn placeholder vào một class nào đó, thêm @extend %extend_name;

    See the Pen Extend by Thang (@thang21) on CodePen.


    NOTE: Sự khác biệt giữa Mixins và Extend/Placeholder: 'Mixins' có thể truyền vào biến nhưng 'Extend' thì không.
    Mixins khá tiện nếu có nhiều biến khác nhau. Tuy vậy, việc viết Mixins sẽ bị lặp lại code nhiều lần nếu biến đó giữ nguyên hoặc không truyền biến gì. Chính vì thế mà ta sẽ dùng Extend và trường hợp này, sẽ khiến cho tất cả class có chung 1 thuộc tính sẽ viết trên cùng 1 dòng.

    See the Pen Mixin vs Extend by Thang (@thang21) on CodePen.

  • Extend

    Tương tự như 'Placeholder', muốn lấy 1 đoạn CSS từ 1 class này sang class khác.
    Để tạo extend: %extend_name {...}
    Khi muốn chèn extend vào một class nào đó, thêm @extend .class_name;

    See the Pen mXdgQY by Thang (@thang21) on CodePen.
